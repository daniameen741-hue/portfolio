# Brand Spec — Danish Ibn-E-Ameen Portfolio

## Color Tokens

```css
:root {
  --bg:      oklch(97.5% 0.003 240);
  --surface: oklch(100% 0 0);
  --fg:      oklch(12% 0.015 240);
  --muted:   oklch(50% 0.018 250);
  --border:  oklch(92% 0.005 240);
  --accent:  oklch(52% 0.18 255);

  --card-bg:  rgba(255, 255, 255, 0.65);
  --secondary-text: #6B7280;
}
```

### Hex equivalents (for image content)
- Primary: #0B0B0B
- Background: #F8F9FA
- Card background: rgba(255, 255, 255, 0.65)
- Border: rgba(255, 255, 255, 0.30)
- Accent: #2563EB
- Text: #111111
- Secondary text: #6B7280

## Typography

- **Display / Headings**: Manrope (Google Fonts), 400–700 weight
- **Body**: Inter (Google Fonts), 400–500 weight
- **Scale**: Large headlines 72px, Section headings 40px, Body 18px, Buttons 16px

## Layout Posture

- Bento Grid layout — 12-column responsive grid
- Cards: 24px border-radius, soft shadow, subtle glass effect (backdrop-filter: blur)
- Hover lift animation on cards (translateY -4px)
- Large white space between sections
- Padding: generous (32–48px inside cards)
- Perfect alignment — grid-based, no floating elements
- Minimal sticky navigation with frosted glass effect

## Image Style

- Large photography with rounded corners
- Editorial, luxury, minimal, high resolution
- 12–16px border-radius on images

## Motion

- Smooth fade-in on scroll
- Slide-up entrance
- Hover lift (scale + translate)
- Micro-interactions on buttons and links
- No flashy or heavy animations
- Transition durations: 0.3–0.5s ease-out
